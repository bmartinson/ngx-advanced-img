Image Optimization

Select an image and optimize it for size if you provide a byte limit.

Open your JS Console to review what's going on after clicking optimize... Have questions? Read the README.
Required. Starting scale of the image overall.
Required. Starting encoding quality.
Optional. Max bytes to optimize the file to. Pass 0 and the file will just be scaled and encoded based on the above parameters. MB to Byte Calculator if you need help
Optional. Max allowable pixels in a given dimension (width/height).
Require, optimization mode. Go check the readme linked above to understand how each works.
If turned off, the optimize will do its best and give you its best. When on, it will fail if the file size can't be achieved.
